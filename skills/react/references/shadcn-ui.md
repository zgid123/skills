# shadcn/ui Integration

## Alias Configuration

Configure `components.json` to install generated shadcn/ui components into a dedicated sub-folder:

```json
{
  "aliases": {
    "ui": "#/components/shadcn-ui"
  }
}
```

This means every component added via `npx shadcn@latest add <component>` lands in `src/components/shadcn-ui/`. The `#/` prefix follows the project's `package.json` imports convention (see the TypeScript imports skill).

## Folder Layout

`src/components/` is split into two sub-folders when using shadcn/ui:

```
src/components/
  shadcn-ui/       # generated by the shadcn CLI — never edit these files manually
    button.tsx
    input.tsx
    dialog.tsx
    ...
  core/            # our design system — thin wrappers over shadcn-ui with barrel export
    Button/
      Button.tsx
      index.ts     # export { Button } from './Button'
    Input/
      Input.tsx
      index.ts
    index.ts       # root barrel: re-exports all core components
```

## `shadcn-ui/` — Generated Components

- Contains only files produced by the shadcn CLI. **Never edit them directly.**
- Treat them as a vendored dependency: re-run the CLI to update, never hand-patch.
- Do not import from `shadcn-ui/` directly in feature code; always go through `core/`.

## `core/` — Design System

`core/` wraps shadcn/ui primitives with the minimum changes required to match the project's design system (tokens, default variants, class overrides). Each component in `core/` follows the standard folder convention.

### Rules

- Import shadcn/ui primitives through the `#/components/shadcn-ui` alias.
- Apply only the changes needed to conform to the design system — no speculative variants or props.
- Export every component through the `core/index.ts` barrel; feature code imports from there.
- Never bypass `core/` to import directly from `shadcn-ui/`.

```tsx
// ✅ core/Button/Button.tsx — minimal wrapper
import { Button as ShadcnButton } from '#/components/shadcn-ui/button';

interface IButtonProps {
  label: string;
  onClick: () => void;
}

export function Button({ label, onClick }: IButtonProps): JSX.Element {
  return (
    <ShadcnButton
      className="font-brand rounded-xl"
      onClick={onClick}
    >
      {label}
    </ShadcnButton>
  );
}
```

```ts
// ✅ core/index.ts — barrel for the entire design system
export { Button } from './Button';
export { Input } from './Input';
```

```ts
// ✅ Feature code imports from core, not from shadcn-ui directly
import { Button } from '#/components/core';

// ❌ Never import shadcn-ui components directly in feature code
import { Button } from '#/components/shadcn-ui/button';
```
